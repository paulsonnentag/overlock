Overlock demo

Pick a tool, pick a colour, and drag inside the white canvas. Each child tool walks up via element.parentComponent until it finds the <paint-canvas> host, sets canvas.dataset.selectedTool on click, reads host.color when drawing, and listens for pointer events on the host.

The page also tries to load tools/bad-line/ — an intentionally illegal tool. The verify-on-load middleware rejects it before mounting; the failure shows up in the red box below while the legal tools mount as normal.
